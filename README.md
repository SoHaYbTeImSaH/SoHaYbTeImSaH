افشاء هویت، اطلاعات و اسرار طفل یا نوجوانان بزه دیده یا در وضعيت مخاطره آمیز .

انتشار هرگونه محتوا در جهت تحریک یا ترغیب به کودک آزاری در فضای مجازی.

برقراري ارتباط با طفل و نوجوان در فضاي مجازي به‌منظور آزار جنسي .

لینک کانال
 1- محتوا علیه عفت و اخلاق عمومی. (

1. اشاعه فحشاء و منكرات. (         )
2. تحریك ، تشویق ، ترغیب ، تهدید یا دعوت به فساد و فحشاء و ارتکاب جرایم منافی عفت یا انحرافات جنسی.   (           )

3. انتشار ، توزیع و معامله محتوای خلاف عفت عمومی. ( مبتذل و مستهجن )  (            )

4. تحریك ، تشویق ، ترغیب ، تهدید یا تطمیع افراد به دستیابی به محتویات مستهجن و مبتذل.   (           )

5. استفاده ابزاری از افراد ( اعم از زن و مرد ) در تصاویر و محتوا ، تحقیر و توهین به جنس زن ، تبلیغ تشریفات و تجملات نامشروع و غیرقانونی. 

(             )

ب ) محتوا علیه مقدسات اسلامی  (               )

1. محتوای الحادی و مخالف موازین اسلامی  (             )

2. اهانت به دین مبین اسلام و مقدسات آن 
(               )
3. اهانت به هر یک از انبیاء عظام یا ائمه طاهرین ( ع ) یا حضرت صدیقه طاهره ( س )   (             )

4. تبلیغ به نفع حزب گروه یا فرقه منحرف و مخالف اسلام 
(           )
5. نقل مطالب از نشریات و رسانه ها و احزاب و گروه های داخلی و خارجی منحرف و مخالف اسلام به نحوی كه تبلیغ از آنها باشد. 
(         )
6. اهانت به امام خمینی ( ره ) و تحریف آثار ایشان 
(           )
7. اهانت به مقام معظم رهبری ( امام خامنه ای )     (           )

 

ج ) محتوا علیه امنیت و آسایش عمومی.  (          )

1. تشكیل جمعیت ، دسته ، گروه در فضای مجازی ( سایبر ) با هدف برهم زدن امنیت كشور. (            )

2. هر گونه تهدید به بمب گذاری.  (            )
3. محتوایی كه به اساس جمهوری اسلامی ایران لطمه وارد كند.   (          )

4. انتشار محتوا علیه اصول قانون اساسی. 
(          )
5. تبلیغ علیه نظام جمهوری اسلامی ایران.     (              )

6. اخلال در وحدت ملی و ایجاد اختلاف مابین اقشار جامعه به ویژه از طریق طرح مسائل نژادی و قومی.  (           )

7. تحریك یا اغوای مردم به جنگ و كشتار یكدیگر.  (             )

8. تحریك نیروهای رزمنده یا اشخاصی كه به نحوی از انحا در خدمت نیروهای مسلح هستند به عصیان ، فرار، تسلم یا عدم اجرای وظایف نظامی. (          )

9. تحریص و تشویق افراد و گروه ها به ارتكاب اعمالی علیه امنیت ، حیثیت و منافع جمهوری اسلامی ایران در داخل یا خارج از كشور.  (           )

10. تبلیع به نفع گروه ها و سازمانهای مخالف نظام جمهوری اسلامی ایران  (           )
11. فاش نمودن و انتشار غیرمجاز اسناد و دستورها و مسایل محرمانه و سری دولتی و عمومی. 
(           )
12. فاش نمودن و انتشار غیرمجاز اسرار نیروهای مسلح.  (          )

13. فاش نمودن و انتشار غیرمجاز نقشه و استحكامات نظامی. 
(          )
14. انتشار غیرمجاز مذاكرات غیرعلنی مجلس شورای اسلامی.  (           )

15. انتشار بدون مجوز مذاكرات محاكم غیرعلنی دادگستری و تحقیقات مراجع قضایی. 
(           )
16. انتشار محتوای که از سوی شورای عالی امنیت ملی منع شده باشد.
(         )
 

د ) محتوا علیه مقامات و نهادهای دولتی و عمومی (        )

1. اهانت و هجو نسبت به مقامات ، نهادها و سازمان های حکومتی و عمومی
(            )
2. افترا به مقامات ، نهادها و سازمان های حکومتی و عمومی. (           )
3. نشراكاذیب و تشویش اذهان عمومی علیه مقامات ، نهادها و سازمانهای حکومتی. (         )
 

ه ) محتوای که برای ارتکاب جرایم رایانه ای به کار می رود (          )

1.انتشار یا توزیع و در دسترس قرار دادن یا معامله داده ها یا نرم افزارهایی كه صرفاً برای ارتكاب جرایم رایانه ای به كار می رود. (               )

2. فروش انتشار یا در دسترس قرار دادن غیرمجاز گذرواژه ها و داده هایی كه امكان دسترسی غیرمجاز به داده ها یا سامانه های رایانه ای یا مخابراتی دولتی یا عمومی را فراهم می كند.  (            )

3. انتشار یا در دسترس قرار دادن محتویات آموزش دسترسی غیرمجاز ، شنود غیرمجاز ، جاسوسی رایانه ای ، تحریف و اخلال در داده ها یا سیستم های رایانه ای و مخابراتی.  (         )

4. آموزش و تسهیل سایر جرایم رایانه ای. (           )

5. انتشار فیلترشكن ها و آموزش روشهای عبور از سامانه های فیلترینگ.    
6. انجام هرگونه فعالیت تجاری و اقتصادی رایانه ای مجرمانه مانند شركت های هرمی.

2. محتوا مجرمانه مربوط به امور سمعی و بصری و مالکیت معنوی. 

1. انتشار و سرویس دهی بازی های رایانه ای دارای محتوای مجرمانه.  2. معرفی آثار سمعی و بصری غیرمجاز به جای آثار مجاز. (           )
3. عرضه تجاری آثار سمعی و بصری بدون مجوز وزارت فرهنگ و ارشاد اسلامی.
